# PokeExplorer

**Project:** IT5103N Advanced Mobile Development Final Project  
**Description:** Interactive AR-enhanced Pokedex for React Native (iOS/Android)

An app that lets you discover, capture, and catalog Pokemon in real-world contexts using geolocation, AR overlays, and device sensors. Inspired by Pokemon Black and White's Unova Pokedex.
